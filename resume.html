This project displays the QR code of the most recent Air Circuit Breaker and Cradle being assembled and tested on the production line in real time. This helps the operator to synchronize the production of Cradle according to its respective ACB.

Farmers Guide

This project is a Realtime data monitoring system for the farmers. This project reads analog data with the help of various sensors from the surrounding atmosphere and uplpoads it to a real time datebase (Google firebase). It displays this data in real time on a web application in both numeric and graphical ways. This project is very useful for data anlysis of any geological area in real time. This project is capable of User Authentication to enable multiuser useability.
